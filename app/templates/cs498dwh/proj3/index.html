Poisson & Mixed Blending

The main idea behind fusing of a image into another is that our eyes perceive the overall gradient of an image more than the intensity. Therefore, as long as we can preserve the gradient of the resultant image, the change in intensities does not make it stand out of the target image. To blend an image using the poisson process, the new intensity values 'v' are to be found using the pixel intensities of the source image "s" and of the target image "t". For each pixel, its four neighbors are checked, if a neighbor is outside the mask region, we use the second part of the equation, using the target image, else we use the source image.

For Mixed blending, we use the gradient in source or target with the larger magnitude as the guide, rather than just the source gradient.
