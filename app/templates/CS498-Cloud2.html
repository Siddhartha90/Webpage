Cloud Computing

This was the other cloud computing class I took, which dealt with the intersection of mobile computing and cloud computing. Wireless networking and cloud computing theory was covered, and a significant portion of the course was devoted to practical mobile software development.
We made an app on Windows platform. (The necessary mobile and cloud computing infrastructure was provided in partnership with Microsoft Research.)

Convo: a chat platform with a social networking essence.

Users could post statuses reflecting what they’ve been upto and if they are planning any events. They could also upload a profile picture for which we used the blob storage.
